# Watercolor App
Convert image to watercolor painting

## Download Source Code
```bash
git clone https://github.com/playatanu/watercolor-app watercolor_app

cd watercolor_app
```
## Install Packages
```bash
pip install -r requirements.txt
```
## Run
```python
streamlit run app.py
```
